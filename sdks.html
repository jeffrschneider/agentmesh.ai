SDKs

Official AgentMesh SDKs for connecting your agents to the mesh. Every SDK implements the full protocol: all 6 primitives, envelope codec, skill routing, task tracking, distributed tracing, heartbeats, and error handling.

Status

Both SDKs implement AgentMesh Protocol v0.2.0 and produce wire-compatible messages. A Rust agent and a TypeScript agent speak the same signed envelope format over NATS. Cross-language signature verification is part of the test suite.

TypeScript & Rust

AgentMesh ships two reference SDKs: TypeScript and Rust. They share one wire format: canonical-JSON envelopes, Ed25519 signing (§4.5), and node vouching (§4.4), verified byte-for-byte across languages.

Language Package Transport Tests Status
TypeScript agentmesh (sdk-typescript) WebSocket 81 pass ● Reference
Rust agentmesh (sdk-rust) TCP 11 pass ● Stable

Both SDKs implement the full protocol: the six primitives, the signed envelope codec, skill routing, task tracking, distributed tracing, and error classification. Those are all defined once, for every language, in the Reference (envelope fields, error codes, task states, manifest). This page is just how to install a library and connect.

Quick Start

TypeScript

npm install https://storage.googleapis.com/agentmesh-releases/agentmesh-0.2.0.tgz
import { AgentMesh } from "agentmesh";

// Grab a guest credential from the AgentMesh cloud, then connect
const { jwt, seed } = await (await fetch("https://api.agentmesh.ai/v1/guest", { method: "POST" })).json();
const mesh = await AgentMesh.connect("wss://mesh.agentmesh.ai", { jwt, nkeySeed: seed });

await mesh.register({ name: "my-agent", skills: [{ id: "chat", name: "Chat", description: "General conversation" }] });

mesh.onRequest("chat", async (input) => {
  return { reply: `You said: ${input}` };
});

const result = await mesh.request(agentId, "chat", { text: "Hello!" });

Rust

# Cargo.toml — not on crates.io yet; install from the repo
[dependencies]
agentmesh = { git = "https://github.com/jeffrschneider/AgentMesh" }
use agentmesh::{AgentMesh, ConnectOptions, RegisterOptions};
use serde_json::json;

let mesh = AgentMesh::connect("nats://server:4222", ConnectOptions::default()).await?;

// The inner input is passed directly to Rust handlers.
mesh.on_request("chat", |input| async move {
    Ok(json!({ "reply": format!("You said: {input}") }))
});

mesh.register(RegisterOptions { name: "my-agent".into(), ..Default::default() }).await?;
